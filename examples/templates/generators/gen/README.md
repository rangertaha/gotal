# Sine

## Overview

This example demonstrates how to generate a sine wave time series using the `gen.Sine` generator from the `gotal` library.

![Sine Wave Example](sine.png)

## Example

```go
package main

import (
	"time"
	"github.com/rangertaha/gotal/pkg/provider"
)

func main() {
	var Tags = map[string]string{
		"symbol":   "BTC-USD",
		"exchange": "BINANCE",
		"currency": "USD",
		"asset":    "BTC",
	}

	// Generate a new series of sine waves
	prices := gen.Sine(time.Second, 100, 0.1, 0, Tags)

	// Display table
	prices.Print()
}
```
